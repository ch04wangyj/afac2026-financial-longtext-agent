# V11：增益、回归与双错三态约束

## 模型修正

变化题不再强制二选一，而是允许：

- `gain`：新答案命中，贡献 `+1`；
- `loss`：旧答案命中，贡献 `-1`；
- `neutral`：旧、新答案都未命中，贡献 `0`。

`neutral` 会在 MILP 中显式把旧、新答案变量固定为 0。这样既能解释 V10 的奇数
净分差，也不会把“尚未观察到第三标签”误当成零信息。

## V10 反馈后的硬结论

- `reg_a_004=AC` 必错，可行状态为历史 `ABC` 或第三标签。
- `fc_a_012=AC`、`fc_a_015=A`、`res_a_011=BC` 均不可能是标准答案。
- `fc_a_012=ACD` 与 `fc_a_015=B` 中恰有一个命中，另一个的新旧答案都错。
- 联合可行只证明候选不与历史总分矛盾，不证明候选就是真实标签。
