  - **B 类（需恢复）**：图像化的数据表/财务表 —— V14 无法解析，是准确率漏点

### 1.2 V14 版面算法现状与边界

| 模块 | 现状 | 已知边界 |
|---|---|---|
| 跨页表格 | `_is_continuation` 仅处理相邻页，要求行宽相同+表头相同/几何连续 | 不处理跨多页、中间隔页、表头变异 |

| 表头层级 | 仅展开"两年度×金额/比例+同比" | 多级表头、合并单元格、嵌套结构未处理 |

### 1.3 赛题约束关键澄清

赛题原文：
> "参赛者需调用 **Qwen 系列模型 API** 完成推理，不得使用任何其他开源或闭源模型替代"
> "文档离线解析阶段的 Token 消耗**不计入**最终 Token 消耗"
> "禁止在任何场景下使用 **embedding** 模型"

**结论**：Qwen-VL 是 Qwen 系列，赛题未禁止。离线解析阶段用 Qwen-VL 提取图像表格文本，Token 不计费。这是合规且高效的优化路径。

## 二、技术路线

### 路线 A：离线 Qwen-VL 图像表格恢复（主攻 B 类）

**核心思路**：在文档预处理阶段，对 B 类图像页调用 Qwen-VL，把图像化表格转成结构化文本，作为 V14 语料的增量补充。

**流程**：
1. 精准定位 B 类页（图像占>50% + 文字层<150字 + 上下文有财务术语）
2. 用 PyMuPDF 把 B 类页渲染成 PNG（300 DPI）
3. 调用 Qwen-VL（qwen-vl-max）提取表格为 Markdown/JSON 结构化文本
4. 对 Qwen-VL 输出做确定性校验（数值/年份/单位完整性检查）
5. 合规的 Qwen-VL 文本作为 `layout_vl_table_row` 增量块接入索引

**Token 预算**：
- B 类页 70 页（财报 38、合同 26、研报 6；二次过滤后真正含数据表的估计 20-40 页）
- 每页 Qwen-VL 调用约 2000-4000 token（图像+输出）
- 总离线 Token 约 140K-280K（**不计入最终 Token**）
- 在线答题 Token 预算保持 V14 水平（~312K）

**合规风险**：
- 赛题说"离线解析阶段 Token 不计费"，Qwen-VL 用于"文档结构化解析"属于此范畴
- 需在提交报告里明确说明：Qwen-VL 仅用于离线图像表格文本提取，不参与在线推理
- 保留 Qwen-VL 调用日志，可审计

### 路线 B：版面算法四项深化（主攻 A 类以外的坐标页）

#### B1. 跨页表格扩展（多页续表）

**现状**：`_is_continuation` 仅处理相邻页，且要求行宽完全相同。

**改进**：
- 扩展到跨多页（page N → N+1 → N+2，链式继承）
- 支持中间隔页（page N 和 N+2 是同一表，N+1 是插图页）
- 表头变异容忍：允许列顺序变化、单位行变化、新增/删除列
- 几何条件放宽：不要求 bbox 精确连续，改为"页面底部+下页顶部"区域匹配

**实现**：`layout_pdf.py` 新增 `_find_continuation_chain`，替代单步 `_is_continuation`。

#### B2. 双栏阅读顺序

**现状**：要求两侧垂直重叠才判定双栏。

**改进**：
- 增加基于文本块 X 坐标聚类的栏检测（不依赖垂直重叠）
- 处理"栏间有跨栏标题/表格"的混合版面
- 处理"左栏短、右栏长"的不对称双栏
- 阅读顺序：左栏从上到下 → 右栏从上到下

**实现**：`layout_pdf.py` 新增 `_detect_columns_by_clustering`，与现有 `_detect_two_column` 并行，取更优结果。

#### B3. 页眉页脚精细化

**现状**：`recurring_margin_ratio=0.28` 检测重复文本。

**改进**：
- 增加基于跨页 X 相似度的页眉页脚检测（不只是完全相同）
- 处理"页码变体"（第 X 页 / Page X of Y / 纯数字）
- 处理"日期变体"（2026 年 X 月 X 日）
- 印章检测：圆形/椭圆形矢量图形区域标记为噪声

**实现**：`layout_pdf.py` 新增 `_detect_recurring_headers_fuzzy`。

#### B4. 表头层级展开

**现状**：仅展开"两年度×金额/比例+同比"。

**改进**：
- 支持三级表头（大类→子类→年份）
- 支持合并单元格（rowspan/colspan 检测）
- 支持嵌套结构（"营业收入"下含"主营业务收入""其他业务收入"）
- 表头继承：跨页续表自动继承完整表头层级

**实现**：`layout_pdf.py` 新增 `_expand_multilevel_header`。

### 4.2 风险与缓解

| 风险 | 等级 | 缓解 |

| Qwen-VL 表格识别错误引入噪声 | 中 | 确定性校验：数值/年份/单位完整性，不合规的丢弃 |

| 跨页链式继承误判 | 低 | 严格几何+表头双条件，宁可漏判不可误判 |
